NeoVim
- Instalado o NeoVim;
- Criado e configurado arquivo init.lua, no diretório ~/.config/nvim. Nesse arquivo é possível adicionar 
  vários parâmetros de configuração;
	- vim.keymap.set('n', '<C-a>', 'ggVG') - configura o Ctrl+a para selecionar todo o texto
	- vim.opt.spelllang = {'pt'}  - adiciona dicionário e verifica em tempo real os termos
	- vim.opt.spell = true - ativa dicinário

- Poderia alterar o .bashrc ou .zshrc para ativar o nvim através do vi, mas não quis;
- Após perceber que estava configurando o vi em vez do nvim, consegui configurar o spelllang para pt. Pra isso adicionei
  duas propriedades ao ./config/nvim;

	
Comando úteis:
	- :%s "texto" = localiza texto
	- vsp "nome_do_arquivo" já aberto. Ele coloca lado a lado, mas não percebi utilidade


Dotfiles
	- Arquivos de configurações comuns do Git, NeoVim, VSCode, bash, etc
	- Criar repositório no Git e Github
	- Mover (mv caminho_do_arquivo_original caminho_do_repositórioLocalGit_dotfiles):
		mv ~/.config ~/git/dotfile
			neste exemplo, o arquivo nvim está dentro de .config
	- Criar link simbólico: ln -s caminho_do_repositórioLocalGit_dotfiles caminho_do_arquivo_original
		ln -s ~/git/dotfiles/ ~/.config
			ao executar o 1° comando deixará de existir o ~/.config, mas na criação do link simbólico é criado
